Attendance Calculator (Java GUI)

A Java-based GUI application developed to calculate attendance percentage and determine eligibility based on predefined criteria. The project demonstrates the practical application of object-oriented programming and GUI development concepts.

Key Highlights:
- Designed an interactive and user-friendly GUI
- Implemented accurate attendance calculation logic
- Applied core OOP principles such as classes and methods
- Improved user experience through clear output and validation

Technologies Used:
- Java
- Java Swing / AWT
- Object-Oriented Programming

Project Structure:
- `AttendanceCalculatorGUI.java` – GUI and user interaction
- `AttendanceClasses.java` – Business logic and calculations

How to Run:
1. Ensure Java is installed on your system
2. Compile the files: javac AttendanceCalculatorGUI.java, AttendanceClasses.java
3. Run the application: java AttendanceCalculatorGUI

Purpose:
This project demonstrates the use of Java GUI programming, OOP concepts, and basic application logic.
